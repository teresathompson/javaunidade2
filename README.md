# ☕ Java Unidade 2

Repositório contendo exemplos, exercícios e práticas desenvolvidas durante as aulas de **Java – Unidade 2**.

O objetivo deste projeto é demonstrar conceitos fundamentais da programação em **Java**, com foco em **Programação Orientada a Objetos (POO)**, organização de código e execução de programas simples. :contentReference[oaicite:1]{index=1}

# 📚 Conteúdo do Repositório

Este repositório contém exemplos utilizados durante as aulas, incluindo:

- Criação de classes em Java
- Construtores
- Encapsulamento
- Métodos getters e setters
- Validação de dados
- Execução de programas via método `main`
- Organização de projetos Java
